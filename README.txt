네이버 스마트스토어 리뷰 크롤러 2.0
---------------------------------------
<App 실행방법>
1. 스마트스토어의 아무 회사의 스토어에 접속(스마트스토어 메인은 안됌) 예제 주소 ->  https://smartstore.naver.com/pgw
2. 개발자모드(F12)를 켜고 Console 메뉴로 변경 
3. app.js 파일 내용을 전체 복사하여 입력후 enter
4. 화면이 새로 갱신되면 성공
(시간을 조금이나마 단축하려면 https://smartstore.naver.com/pgw 주소를 바탕화면에 바로가기를 만들어서 사용할때마다 접속후 파일 전체 복사 붙여넣기)

<App 사용방법>
1. 스마트스토어의 찾고자 하는 상품페이지에 접속
2. 개발자 모드(F12)를 켜고 Network 매뉴로 변경
3. 상품의 리뷰 매뉴 클릭후 최신순으로 정렬
4. Network 매뉴에 새로운 파일들이 갱신되는 것을 볼 수 있는데 filter 검색창에 paged 입력시 paged-reviews?블라블라 파일이 보인다.
5. (하나가 아니라면 가장 밑에 있는) 파일 우클릭 -> Copy -> Copy link address 클릭
6. 복사한 주소를 url 박스에 입력 , 날짜의 경우 예시로 2020-09-01을 입력시 오늘날짜부터 입력날짜 사이의 리뷰들만 탐색
7. 탐색된 리뷰 수는 마우스 클릭으로 복사 가능

*제한사항* 
- 페이지가 갱신된 이후 나는 오류는 스마트스토어 오류로 크롤링에는 아무 지장이 없음
- 많은 리뷰를 긁어올때 아주 가~끔 예외처리에서 걸려 댓글이 안가져와지는 현상이 발생하는데 새로고침이후 현상해결 쌉가능
- 날짜 입력후 바로 버튼 클릭시 반응이 살짝 느린데 크롤링에는 아무 지장이 없음
